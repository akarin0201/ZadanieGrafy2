## Obsługa programu:

Wykonać plik bin/Debug/ZadanieGrafy2.exe  
Plik można wykonać z flagą -f, aby json grafu został pobrany z wyszczególnionego pliku:  
```ZadanieGrafy2.exe -f sciezka/do/pliku.json```

Aby wyświetlić informacje o obliczeniach plik można wykonać z flagą -d  
(przy dużych grafach wypisanie wszystkich informacji zajmuje dużo czasu, i i tak nic z tego nie wiadomo)
